## AI Credit Risk Scoring & Loan Decision System

XGBoost-based loan approval prediction system with 98.24% accuracy.

## Tech Stack
XGBoost, scikit-learn, pandas, NumPy, Matplotlib, Seaborn

## Features
- EDA visualizations (income distribution, credit score analysis)
- XGBoost classifier with 200 estimators
- Confusion matrix and feature importance analysis
- Manual applicant input for real-time prediction

## Results
- Accuracy: 98.24%
- AUC-ROC: 0.91
- Dataset: HDFC loan approval dataset

## How to Run
1. Upload `loan_approval_dataset.csv` when prompted
2. Model trains automatically
3. Enter applicant details for prediction
